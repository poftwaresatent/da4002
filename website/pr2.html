ITADS Project 2
sequence alignment

Finding a good alignment between two sequences of symbols (characters) is a commonly encountered problem in various domains. The Needleman-Wunsch algorithm presented in lecture 6 efficiently finds such optimal alignments, and this project is about implementing it. The excellent article on Wikipedia explain this algorithm in more detail.

The deadline for handing in the source code and the report is Friday, October 18, 2013, at 18h00. During the three lab sessions on Oct 11, Oct 15, and Oct 18, you can get help from the lecturer and assistant in the Unix room B231c. Please do take advantage of these valuable hours to advance on your project.

You are provided with the following files as a starting point. Read them carefully, as they contain a lot of very relevant and detailed information.

  • Project 2 assignment in PDF format. Follow the given instructions very carefully.
  • example-matrix-a.c and example-matrix-b.c provide starting points for implementing matrices in C.
  • example-argument-parsing.c provides code for some of operations you will need to perform on the provided command line arguments.
