ITADS Exercise 8
dynamic programming

Exercise 8.1: equipment replacement revisited

Read the solution of today's second group activity (the one about equipment replacement). Then answer the following questions to make sure you understand the solution.

  1. List all the optimal solutions.
  2. Let the maintenance costs be higher: m1=100, m2=110, m3=200. Then:
    • Redo the computations for all stages.
    • What is the optimal cost after this change?
    • What is the new optimal solution (there is only one)?
  3. Optional (time intensive): let the maximum age of a machine be 4 instead of 3, and use the following maintenance costs and trade values: m1=60, m2=80, m3=120, m4=130, s1=800, s2=600, s3=500, s4=400. Then:
    • Redo the computations for all stages.
    • What is the optimal cost after this change?
    • Give the two new optimal solutions.

  1. Write a program called change that executes the change-making algorithm for n={1, 2, 3, ... 20} and prints the resulting table to the terminal. The output should for example look like this:
