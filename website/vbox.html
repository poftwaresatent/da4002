ITADS Virtual Appliance

In case you want to work on the exercises and projects at home, but you only have a Windows machine, here you can get a virtual machine image with a bare-bones Linux system on it.

  1. Install VirtualBox by downloading the appropriate file and following the instructions on www.virtualbox.org
  2. Download the ITADS virtual appliance file: itads-2012-09-27.ova
  3. Import the ITADS appliance into VirtualBox:
    • Launch VirtualBox.
    • Go to "File -> Import Appliance..."
    • Select the itads-2012-09-27.ova file.
    • Finish the import (maybe adjust the settings a bit, but the defaults should be fine).
  4. Launch the itads virtual machine inside VirtualBox. This will boot the virtual machine and present you with an interface very similar to the one on the computers in the exercise room.

This gives you a small Linux system (based on the Debian distribution) with a setup that is very similar to the workstations in the University's UNIX room B231c. In case the ITADS appliance ever asks you for a user name, password, or administrative password, they are all very simple to remember:

user name:
itads
password:
itads
root (administrator) password:
itads
